# 박지영의 3개월 바운더리 대시보드

키즈러닝랩 박지영 대표의 **2026.06.23 → 2026.09.23 (3개월 / 92일 / 12주)** 바운더리 트래킹 대시보드.

> **한 줄 바운더리:**
> 3개월 동안 `<왜 우리는 같이 뛰는가>` 시리즈 릴스 36개 (주 3개) 올려서,
> 엄마들 입에서 "어디 가면 같이 달릴 수 있나요?" 나오게 한다.
>
> **선 넘으면 아웃이야. 오케이?**

## 주요 기능

- **KPI 카드 3개** — 업로드 릴스(N/36), 경과 일자(N/92), 측정 누적 카운트
- **3개월 캘린더** — 주별 7열 그리드, 업로드 회차는 warm red 도트, 오늘 강조, 빈 날짜 클릭 시 폼으로 점프
- **콘텐츠 추가/편집 폼** — 회차 자동 생성, 인스타 링크, 메모
- **콘텐츠 목록 테이블** — 시간순 정렬, 검색(제목/메모), 편집/삭제
- **IN / OUT 박스** — 바운더리 상기용 (소정샘 인터뷰 그대로)
- **주간 측정 입력** — 매주 일요일 결산용, DM/댓글·"어디 가면 같이 달릴 수 있나요?"·노트
- **데이터 가져오기/내보내기** — `boundary_data_YYYYMMDD.json`
- **LocalStorage 자동 저장** — 브라우저에만 보관, 외부 전송 없음
- **반응형** — 데스크탑 3열 / 모바일 단일 컬럼

## 사용법

1. https://nangmanwife.github.io/boundary-dashboard/ 접속
2. 달력에서 오늘 날짜 클릭 → 회차/제목/메모/인스타 링크 입력 → 저장
3. 매주 일요일 → "주간 측정" 폼에 그 주 DM·시그널 카운트 기록
4. 데이터 백업: 우상단 "데이터 내보내기" → JSON 파일로 저장
5. 다른 브라우저/기기로 옮길 때: "데이터 가져오기" 로 JSON 불러오기

## 기술 스택

- Vanilla HTML / CSS / JS (단일 파일 없음, 3개 분리)
- 외부 라이브러리: Pretendard CDN 1개
- 빌드 도구 없음 (정적 파일 그대로 GitHub Pages 배포)
- 데이터 저장: 브라우저 `localStorage` (key: `boundary-dashboard:v1`)
